Filtering

You can filter the todo list by user and building using the input fields provided. Enter the user or building name in the respective fields, and the table will be updated accordingly.

Adding a New Row

To add a new row to the todo list, click the "Add New Row" button. A new row with default values will be added to the table.

Inline Editing

You can edit the values in each cell by clicking on them. The "Name," "Username," and "Building" fields are editable. For the "Status" field, a dropdown will appear when clicked, allowing you to select a new status. The "Save" button will appear only if any value in the row has been changed.

Saving Changes

After editing a row, click the "Save" button to save the changes. For new rows, the "Save" button will also appear after clicking "Add New Row," allowing you to save the new record.
